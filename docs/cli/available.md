# cortex available

Show all available skills in the repo.

## Usage

```
./cortex available
```

## Description

Discovers all `SKILL.md` files under `src/skills/` and `external/` and prints each skill's relative path and description. This shows what can be installed, regardless of whether it is currently installed. External suite skills are listed in a separate section.

## Examples

```bash
./cortex available
```

## Options

None.
