# README Dashboard Update Contract — Design Layer

After producing a design document, update the subsystem's `README.md` dashboard.

## Location

The README lives at: `docs/{subsystem}/README.md`.

## Update Procedure

1. **Status Matrix** — Set the Design cell for this phase to `[Done]({filename})`
2. **Carry-Forward Contracts** — If this design doc introduces new Part B CONTRACT entries that downstream phases will build on, add them to the Carry-Forward Contracts table
3. **Last updated** — Set to today's date

## Status Values

- `—` = not started
- `[Done]({filename})` = completed, with link to document
