# README Dashboard Update Contract — Test Docs Layer

After producing a test planning document, update the subsystem's `README.md` dashboard.

## Location

The README lives at: `docs/{subsystem}/README.md`.

## Update Procedure

1. **Status Matrix** — Set the Test Docs cell for this phase to `[Done]({filename})`
2. **Last updated** — Set to today's date

## Status Values

- `—` = not started
- `[Done]({filename})` = completed, with link to document
