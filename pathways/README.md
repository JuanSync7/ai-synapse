# pathways

Named bundles of synapses — each YAML file defines a combination of skills, agents, protocols, and tools to install together. Run `./cortex pathway install <name>` to load a pathway.

See `taxonomy/PATHWAY_TAXONOMY.md` for naming conventions and harness values.
