# README Dashboard Update Contract — Implementation Docs Layer

After producing implementation docs, update the subsystem's `README.md` dashboard.

## Location

The README lives at: `docs/{subsystem}/README.md`.

## Update Procedure

1. **Status Matrix** — Set the Impl Docs cell for this phase to `[Done]({filename})`
2. **Carry-Forward Contracts** — If Phase 0b introduces new interfaces that downstream phases will build on, add them to the Carry-Forward Contracts table
3. **Last updated** — Set to today's date

## Status Values

- `—` = not started
- `[Done]({filename})` = completed, with link to document
