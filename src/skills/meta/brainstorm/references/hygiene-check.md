# Hygiene Check

Quick 1-turn scan to combat context window attention decay. Loaded at [H] on ~10 turn cadence. This is NOT a full lens rotation — it's a focused refresh of accumulated state.

---

## Procedure

1. **Read the notepad** — full read, not skimming
2. **Scan for stale open threads** — items marked Open in Zone 2 that have actually been resolved in discussion but not updated
3. **Check for forgotten threads** — threads discussed in conversation but missing from Zone 2
4. **Check for drifted decisions** — have any thread resolutions been implicitly contradicted by later discussion?
5. **Verify lens rotation state** — does the Lens Rotation State table in Zone 1 match actual progress?
6. **Quick summary** — surface what's stale, forgotten, or drifted

---

## Cadence

- **During Phase B:** ~every 10 turns sets a flag. On the next user reply, process the reply normally (update notepad, note findings), then run [H] before composing the next coaching response. Never interrupt mid-exchange — finish the current turn's work first, then checkpoint.
- **Before Done Signal:** Mandatory final hygiene check (part of the final Circle-back)

---

## Duration

1 turn. If major issues surface (stale decisions, forgotten threads, contradictions), route back to the appropriate [B:*] move for focused attention. Hygiene identifies problems; it doesn't fix them.
